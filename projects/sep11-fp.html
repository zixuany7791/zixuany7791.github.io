RiceGrades

The "Freedom Project" for SEP11 is a year-long project all about making something using JavaScript along with third-party JS tools.

For this project, I chose to independently study Firebase in order to help me make RiceGrades, a grading website.

One of the challenges I had during the process of making RiceGrades is that when I am trying to use my knowledge of firebase into RiceGrades, sometimes it doesn't work because it wouldn't be compatible with react so you would have to code in a way that it would work in both firebase and react.

For example, I try to add a role for login so that users can just click on it to assign roles but when we redirect them to the page they needed to be after login in, they didn't go where they are suppose to be because the value hasn't rendered yet.



To solve this solution, my friend said that you can use useState to solve the problem because useState is a react function that allows you to re-render so that the arrow function will return the user's role instead of event.target.value


My takeaway from this project is that you really need to not procrastinate because when you have all the time you need to do this project, you decided to relax a little bit and play games because I am not in a rush. But then when you actually need to do this project because it is about to hit the deadline, you have 2-3 projects from other class to do therefore it almost becomes impossible to finish all of them on time.
