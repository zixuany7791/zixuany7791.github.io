Freedom Project

The “Freedom Project” for SEP10 is all about researching the topic that you liked and practicing brainstorming technologies for future needs.

For my project, I am making an informational website that will shows summaries of some technologies that are used in construction using Jekyll and Bootstrap.

One of the challenges I have is that since I have never make a sidebar before, I will have to think how I can make this. I found an example on the internet, but it doesn’t have the scrollspy effect so I have to add a div element for the entire page for the scrollspy to work. 


                <div class="container-fluid" data-bs-spy="scroll" data-bs-target="sidebar" data-bs-smooth-scroll="true" class="scrollspy-example-2" tabindex="0">

Result:

One takeaway I have is to have a better time management. I have spent way too much time trying to figure out how to have two scrollspy because I want to have two different navigation tools for different sizes of screens. But because in order for scrollspy to work, you need each section to have an id and have all the navigation tools to have the id on the hyperlink, this apparently led to scrollspy not working. When I ask Mr. Mueller for help, he told me to finish the presentation and elevator pitch first then come back. However, I didn’t really listen to his advice and continued working on it completely and thought that I would be fine. After a long time playing around with liquid and javascript, I still couldn’t figure out why it wouldn’t work, eventually falling behind the schedule and I didn't prepare my presentation and my elevator pitch for this project on time. Therefore in the future, I should consider the priorty first so I know what I should get done first before continue working on a feature that is not important to the website.

After finishing this freedom project, I would like to discover more about javascript so next time I have a better knowledge and might less time overcoming the challenge.
